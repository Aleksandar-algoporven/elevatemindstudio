# ElevateMindStudio Brand Assets

## Final Logo Pack

Final logo assets are stored in two places:

- Web public assets: `apps/web/public/brand/elevatemind-final`
- Documentation archive: `docs/brand/elevatemind-final`

Use these files by default:

- App/favicon icon: `elevatemind-icon-100x100-transparent.png`
- Square app icon: `elevatemind-icon-512x512-transparent.png`
- Social/app store icon on bone background: `elevatemind-icon-1024x1024-bone.png`
- Website horizontal logo: `elevatemind-logo-horizontal-transparent.png`
- Compact stacked logo: `elevatemind-logo-stacked-transparent.png`
- Icon-above layout: `elevatemind-logo-icon-above-transparent.png`
- Wordmark only: `elevatemind-wordmark-transparent.png`
- Preview sheet: `elevatemind-logo-asset-pack-preview.png`

## Usage Rules

- Use the brain icon for favicon, small app marks, and OAuth/app review screens.
- Use the horizontal logo in website navigation and landing hero sections.
- Use the stacked or icon-above version only when the layout is square or centered.
- Keep the gold mark on light, bone, or neutral backgrounds.
- Do not mix these final assets with the older extracted logo files unless we are comparing historical versions.
